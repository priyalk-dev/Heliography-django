- Category-wise image gallery

- Future: Wishlist, Cart, and Payment system (ecommerce features)

##  Tech Stack
- **Backend:** Django, Python
- **Frontend:** HTML, CSS, Bootstrap
- **Database:** SQLite 
- **Version Control:** Git & GitHub
